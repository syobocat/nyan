既知の不具合

  • 「ます」がうまく変換されない
    「上一段活用動詞+ます」と「五段活用動詞+ます」の区別がつけられなかったため放置しています。
    「下一段活用動詞+ます」は変換されます。
  • イ音便、撥音便が文末に来るとおかしくなる
    「イ音便/撥音便+過去の助動詞『だ』」と「断定の助動詞『だ』」の区別がつけられなかったため放置しています。
これらの問題を解決するためには形態素解析が必要ですが、実装の方法がわからないため解決されることはないと思います。
許してにゃん。
